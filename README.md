# Customer Churn Deep Dive + Action Plan

## Business Problem
A subscription-based company is losing customers, but leadership does not know why. The goal of this project is to analyze churn patterns, identify the main drivers of churn, and recommend actions to reduce customer loss.

## Objective
Use SQL, Python, and Power BI to answer:

- How has churn changed over time?
- Which customer segments are churning the most?
- What behaviors indicate a customer is likely to churn?
- Are high-value customers at risk?
- What retention actions should the company take?

## Tools
- SQL
- Python
- Power BI
- GitHub

## Expected Deliverables
- SQL churn analysis
- Python exploratory data analysis
- Power BI dashboard
- Churn risk segments
- Churn action playbook

## Dataset
This project uses a public SaaS churn dataset. The original dataset documentation is included in `data/raw/dataset_README.md`.
